# Salary-prediction-regression
Salary prediction machine learning project using linear regression built in Google Colab as part of the IBM Virtual Internship.

The model ask you the experience and gives the predicted salary output.

##Tools Used:
Python
Pandas
Scikit-learn
Google Colab

Algorithm:
Linear Regression


## Methodology
The project follows a machine learning workflow to build a salary prediction model.
1. Data collection
2. Data processing
3. Feature salection
4. Data splitting
5. Model training
6. Prediction
7. Model evaluation

## Results
The linear Regression model was successfully trained using the dataset. 
The model was able to predict employees salaries based on the input features.

The predicted salary values were compared with the actual salary values from the test dataset.

The model achived a satisfactory performance with a good score, indicating that the model can effectively estimate salaries based on the given features.

The visualization of regression line shows a positive ralationship between the years of experience and salary, meaning that salary increases as experience increase.
